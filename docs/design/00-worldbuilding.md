# 世界观总纲

## 时代背景

架空南宋末年（1270 年前后）。蒙古铁骑压境，朝廷风雨飘摇。武林中人表面议论江湖恩怨，暗地里各怀心思——有人想救国，有人想趁乱夺权，有人只想保住自家门派一亩三分地。

玩家扮演一名初入江湖的无名少侠，身世成谜。三十日内，你将在中原、江南、塞北、西域、蜀中五地辗转，结识各路人物，做出一个个无法回头的抉择。

## 核心设定："一念"

"一念成佛，一念成魔。"

本作的核心哲学是**没有纯善路线**。每个抉择都同时影响四维好感度（信任 / 亲密 / 敬畏 / 忌惮）。你救了一个人，可能让另一个人忌惮你；你展现实力让人敬畏，可能同时让亲近的人感到距离。玩家必须接受"有得必有失"的设定，在矛盾中寻找自己的江湖路。

## 五大地理区域

| 区域 | 气氛 | 核心门派 | 江湖传闻 |
|------|------|----------|----------|
| **中原·嵩山** | 正道中枢，规矩森严 | 太虚剑宗 | 武林盟主大会将至，各派暗流涌动 |
| **江南·烟雨** | 暗潮涌动，温柔乡即英雄冢 | 听雨阁 | 江南首富离奇暴毙，牵出惊天阴谋 |
| **塞北·朔风** | 苍凉壮阔，军民一心 | 铁骑营 | 边关告急，蒙古细作混入中原 |
| **蜀中·云雾** | 避世隐秘，医毒不分家 | 药王谷 | 禁药"断魂散"流入江湖，源头指向蜀中 |
| **西域·黄沙** | 神秘危险，异域风情 | 焚天教 | 魔教东进，传闻教主已至中原 |

        │  朝 廷   │ ← 暗卫「影司」渗透江湖

    │ 名义共主

│   │              │

(暗)    (中立)    (敌对)

**朝廷·影司**：皇帝直属暗卫组织，表面维护朝廷体面，实则操纵武林格局以利蒙古战局。玩家若选择入朝，将揭开朝廷"以江湖制江湖"的棋局。

**武林盟**：名义上的武林共主组织，实则各派角力场。当前盟主空缺（前任暴毙），三十日内的盟主大会是主线时间锚点。

**焚天教**：被正道视为魔教，实则教中有大量被朝廷迫害的前忠臣。正邪之分在这里模糊化——玩家深入了解后会发现"魔教"未必是恶。

## 时间线结构

整个游戏发生在 **30 个游戏日内**，每一天分为 4 个时辰：

| 时辰 | 对应现实 | 特点 |
|------|----------|------|
| **卯时**（晨） | 05:00-09:00 | 晨练、偶遇、信息收集 |
| **午时**（昼） | 11:00-15:00 | 拜访、任务、战斗 |
| **酉时**（暮） | 17:00-21:00 | 社交、酒馆、情报交换 |
| **子时**（夜） | 23:00-03:00 | 潜行、密会、危险事件 |

玩家在每个时辰必须选择一个行动，不同选择导致不同的剧情分支和好感度变化。

## 核心冲突线

1. **武林盟主之争**（主线）：前任盟主之死真相扑朔迷离，五大门派各有人选。玩家的选择将决定谁能登上盟主之位——或者盟主制度本身是否存在。

2. **朝廷暗卫渗透**（暗线）：影司以"维护社稷"为名操纵武林，玩家在调查盟主之死时会逐步发现朝廷的棋局。是否揭穿？揭穿后又能如何？

3. **身世之谜**（个人线）：玩家角色的身世与十年前的"血雨林"灭门惨案有关。这条线索串联起多个 NPC 的过去，是部分结局的关键钥匙。

## 情感基调

整体氛围：**苍凉中带温热**。不是纯粹的打打杀杀，而是在乱世中寻找人与人之间的羁绊。每个 NPC 都有自己的苦衷，没有绝对的反派。玩家最终面对的不是"打败大魔王"，而是"在这乱世中，我选择成为什么样的人"。

## 设计原则

- **没有完美结局**：每个结局都有遗憾，但也有温暖
- **抉择不可逆**：不存在 save/load 机制（SPR），每个存档是独一无二的
- **信息不对称**：玩家在不同时机获得不同信息，很多真相需要二周目才能拼全
- **NPC 有自主性**：好感度低时 NPC 会主动疏远甚至敌对，不会等待玩家
